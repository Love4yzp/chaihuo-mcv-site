# 库尔勒折返点

## 在地遥测

- 地形: 塔里木盆地北缘，孔雀河绿洲与天山南麓过渡区
- 阶梯: 第二级阶梯
- 气候: 暖温带大陆性干旱气候
- 极境挑战: 从南疆绿洲线折返北上时的长距离转场、补给与节奏调整

## 在地共创

- 折返转场
- 北上路线衔接

## 现场记

从阿克苏折返库尔勒，作为转向北疆、前往石河子的路线衔接点。
